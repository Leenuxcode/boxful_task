vscode로 작업했습니다.

사용 언어 = html, css(sass), javaScript(jQuery), php, MySQL



모든이미지는 - https://www.boxful.kr/ko/의 페이지 보기에서 다운로드해서 사용했습니다.

boxful_main.html 파일로시작해서 로그인 버튼을 누르면 로그인 페이지로 이동합니다.


요구사항 
    -- 반응형 웹페이지 --
        PC와 모바일버전 제작 완료했습니다.
        태블릿 사이즈와 전체적인 반응형은 저의 작업 속도가 느려서 완성하지 못했습니다.

    -- 슬라이드 --
        화면의 크기를 변경했을 때와
        문서가 준비되었을 때 실행됩니다.

        PC에서 모바일크기로 줄어들 때 슬라이드의 정보 순서가 변경되는 부분은 
        jQuery의 $().html()메소드를 이용해서 변경했습니다.

    -- 회원가입과 로그인 --
        안내해주신 DB에 Table 생성을 못해서 ( 하루동안 구글링을 통해 시도해보았지만 )
        기존에 작업하던 phpmyadmin을 이용해서 작업했습니다.
